Portal Imobiliário

Neste projeto, foram desenvolvidos dois sites para uma Imobiliária ficticia.
No primeiro portal, o anunciante cadastra seu imóvel com informações como valor, venda ou locação.
O outro portal é destinado para as pessoas interessadas em procurar um imóvel, nele é possível a busca
e o site retorna com os imóveis disponíveis, com todas as informações já cadastradas pelo anunciante.

Os sites foram desenvolvidos na Linguagem JavaScript, junto com HTML, CSS e SQL.

Portal de Candidatura

Neste projeto, foi simulado um portal de candidatura para uma empresa de tecnologia, nele é possível

O site foi desenvolvido somente em HTML, CSS e Bootstrap, para práticas de desenvolvimento Web.
